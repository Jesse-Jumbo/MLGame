# PyGameView

## 資料類型

### Line

```json
{
  "type": "line",
  "name": "name_of_line",
  "x1": 100,
  "y1": 150,
  "x2": 200,
  "y2": 250,
  "width": 3,
  "color": "#FFFAAA"
}
```
